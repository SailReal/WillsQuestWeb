# Elements
* Game
  * n players
  * 5 rounds (with choosable category)
* Round
  * Category
  * timeout
  * n questions
  * abortable
* Question
  * Category
  * question
  * n (max. 4) answers of variable type + don't know
  * Optional: timer
  * difficulty (with point mapping)
* Player
  * Name
  * Points per round (and total)
* Category
  * Name
